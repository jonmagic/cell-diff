# CellDiff

CellDiff is a utility for comparing versions of spreadsheets and getting the differences between them at the cellular level.

## Usage

```bash
gem install cell-diff
```

```ruby
require "cell-diff"

s = CellDiff.open("version1.csv", "version2.csv")
s.diff
```
